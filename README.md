# Cee-jaym13 Svt-av1-hdr

## How do I install these formulae?

`brew install cee-jaym13/svt-av1-hdr/<formula>`

Or `brew tap cee-jaym13/svt-av1-hdr` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cee-jaym13/svt-av1-hdr"
brew "<formula>"
```
